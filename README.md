# 🌾 AgriChain - Smart Agriculture Management System

## 📌 Overview
AgriChain is a Spring Boot REST API project designed to manage farmers, fields, crops, distributors, and irrigation schedules.

## 🚀 Features
- CRUD operations for all entities
- RESTful API design
- Swagger UI for API testing
- MySQL database integration
- Layered architecture (Controller, Service, Model)

## 🛠 Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger (OpenAPI)

## 🔗 API Documentation
http://localhost:8080/swagger-ui/index.html

## ▶️ Run Project
mvn spring-boot:run
